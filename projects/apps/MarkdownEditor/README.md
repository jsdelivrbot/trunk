# Markdown Editor

Simple and easy to use offline markdown editor.

## 3rd party

This app is based on these parts:

* [SimpleMDE](https://simplemde.com/)
* [Electron](https://electron.atom.io/)
* [Font Awesome](http://fontawesome.io/)
* [Bootstrap](https://getbootstrap.com/)
* [Antu text-x-markdown](https://commons.wikimedia.org/wiki/File:Antu_text-x-markdown.svg)
